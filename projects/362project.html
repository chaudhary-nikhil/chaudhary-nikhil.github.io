362 Project

Created the game Battleship using 2 STM32F091X microcontrollers. Implemented PWM for game audio, SPI for the TFT LCD, EEPROM via I2C to store high score data, and UART to communicate game status between the 2 MCU's.

Worked with Kshitij Shah, Atharva Bhide, and Akshath Raghav.
